### Как применять?

**Перед каждым уроком:**
1. Прочитать exam_rules.md (правила экзаменов)
2. Вспомнить слабые места студента (student_weak_points.md)
3. Продумать теорию, практику и Challenge задание

1. Обновить project_devops_training.md с детальным прогрессом

3. **ОБНОВИТЬ weekN_cheatsheet.md с материалом урока**
4. Провести качественный code review

1. Обновить project_devops_training.md
2. Обновить student_weak_points.md (если были ошибки)
3. **Обновить weekN_cheatsheet.md с материалом урока**
4. НЕ просто "запоминать", а ЗАПИСЫВАТЬ в файлы!
